# home2

название: управление рецептами

описание:  в этом проекте создаем систему, которая управляет рецептами. Ты их можешь создавать, добавлять, масштабировать и генерить список покупок. За это отвечает 4 класса. 

использование: для того чтобы запустить тесты нужно скачать pytest делается так pip install pytest или pip3 install pytest. А основной код, если хочешь без тестов, можно запустить с помощью print и какая-то переменная, которую ты написал выше.
